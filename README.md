# C++ 基础学习项目

本项目包含13个C++源文件，涵盖从基础语法到函数重载的核心概念。

---

## 文件说明

### test.cpp
**作用**：第一个C++程序，输出问候语  
**运行结果**：
```
Hello,C++!
```

---

## 编译与运行

使用g++编译单个文件：
```bash
g++ -o test test.cpp
./test
```

或使用IDE（如VS Code、Visual Studio）直接运行。
